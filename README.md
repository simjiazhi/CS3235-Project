# CS3235-Project

## User's instructions so far

* Please ensure you have Java version 9.0.4 and above installed
* For mac users, please download airport through terminal
    * sudo ln -s /System/Library/PrivateFrameworks/Apple80211.framework/Versions/Current/Resources/airport /usr/local/bin/airport
	
## Developer's instructions so far

* Clone repo into a CS3235 Folder within your WAMP / LAMP / XAMP
    * e.g. D:\Bitnami\wampstack-7.1.29-0\apache2\htdocs\CS3235
* Add this pdf to your htdocs folder: https://www.comp.nus.edu.sg/~hugh/presentations/cs3235/topic5/Lect5Comms.pdf
    * e.g. D:\Bitnami\wampstack-7.1.29-0\apache2\htdocs\CS3235\Lect5Comms.pdf
* Please enter your own bssid for testing in the approved.bssid file in the /scripts folder
